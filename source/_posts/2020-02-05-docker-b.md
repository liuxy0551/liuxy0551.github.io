　　这两年 Docker 比较火，很多大厂都在使用。这里记录一下常用的指令。
<!--more-->


#### 1、查看镜像列表

``` shell
docker images
```

#### 2、列出正在运行的容器

``` shell
docker ps 或 docker container ls
```

#### 3、列出所有容器

``` shell
docker ps -a 或 docker container ls -a
```

#### 4、删除镜像/容器

``` shell
docker rmi -f [image_id]
docker rm -f [container_id]
```

#### 5、停止/开始容器

``` shell
docker stop [container_id]
docker start [container_id]
```

#### 6、停止所有容器容器

``` shell
docker-compose stop
```
