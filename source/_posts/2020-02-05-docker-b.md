　　这两年 Docker 比较火，很多大厂都在使用。这里记录一下常用的指令。
<!--more-->


#### 1、查看本地镜像列表

``` shell
docker images
```

#### 2、列出正在运行的容器

``` shell
docker ps 或 docker container ls
```

#### 3、列出所有容器

``` shell
docker ps -a 或 docker container ls -a
```

#### 4、删除某个镜像

``` shell
docker rmi -f [image_id]/[image_name]
```

#### 5、删除某个容器

``` shell
docker rm -f [container_id]/[container_name]
```

#### 6、停止/开始某个容器

``` shell
docker stop/start [container_id]/[container_name]
```

#### 7、停止所有容器

``` shell
docker-compose stop
```
