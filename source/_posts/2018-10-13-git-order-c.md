　　git 常用命令系列随笔会记录一些用到的常见命令，这里记录一下`修改上次的 commit`：
<!--more-->


###  1、修改内容和上次的 commit 合并

``` shell
git add .
git commit --amend
```

###  2、只修改上次的 commit message

``` shell
git commit --amend
```
