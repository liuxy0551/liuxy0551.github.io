&ensp;&ensp;&ensp;&ensp;有时候项目中需要需要老版本的 node 支持，所以记录一下如何管理多个 node 版本。

<!--more-->



&ensp;&ensp;&ensp;&ensp;[node 以往的版本](https://nodejs.org/zh-cn/download/releases/)

### 一、管理工具 —— n

&ensp;&ensp;&ensp;&ensp;通过 npm 安装`n`：
```shell
npm i -g n
```


### 二、常用命令

#### 1、安装、激活版本：
&ensp;&ensp;&ensp;&ensp;只需执行`n <version>`即可安装对应版本的 node。如果`<version>`已经安装（通过 n），n 将激活该版本。
```shell
sudo n 11.15.0
```

#### 2、查看当前安装的版本：
```shell
n ls
```

#### 3、选取已安装的版本：
&ensp;&ensp;&ensp;&ensp;键盘上下键选择版本后回车确认
```shell
n
```

#### 4、运行脚本时选择版本：
```shell
n use 11.15.0 app.js
```

#### 6、删除版本：
```shell
n rm 0.9.4
n rm v0.10.0
```
或：
```shell
n - 0.9.4
```
